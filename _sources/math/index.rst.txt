Math
=====================

Equations written in LaTeX
----------------------------------

Some equation examples:

.. math::

   \frac{\partial f(x, y)}{\partial x} = 0

.. math::

   {\cal F}(x) = \sum_{i=0}^{N} a_i \exp(ikx)

Inline math is also possible, e.g. :math:`\sum_{i=0}^{N} a_i = 0`.
